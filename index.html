Quake II - 3D Models

3D Models from TSPG contains versions 1.0 and 1.1
3D Models from Mod DB contains version 1.0 only

The full ZIP file from Mod DB includes three pk3 files:
* The 3D Models theme
* An optional addon "Guards Always Drop"
* An optional addon "Performance Booster"

"Guards Always Drop is an small addon that basically replaces the standard Shotgun Guard and Machine Guard with one that will always drop its gun, unlikely vanilla Q2 3D Models that will have an small chance to drop its weapon for faithfulness purposes, the addon is of course, for balance purposes as I am sure some people may disagree with the fact that these enemies cannot drop their weapons. However, keep in mind that the Infantry / Enforcer (dude with the chaingun in his hand) will still have only the chance to either drop bullets or nails, it will barely change the usual flow of the map, I assure you."
-- From the Readme.txt of the Mod DB zip file of the theme

TSPG Link

"Performance Booster, as its name explain, it helps to improve the performance by replacing the 3D projectiles with their 2D counterparts from QC:DE and also it will remove the gibs that spawn upon their XDeath animation. Will it look silly? Yes, it will. Will it help to boost the performance? Yes, it will, and by a ton, however I don't expect it to work on mobile devices cause obviously the 3D models of each enemy are not replaced (which is basically the main atraction of this mod), for mobile, just play with either QC:DE Quake 2 Theme or Quake 2 Rebalance (for a more faithful experience), since that is the main reason why those exist."
-- From the Readme.txt of the Mod DB zip file of the theme
